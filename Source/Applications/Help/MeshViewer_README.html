Mesh Viewer Application Release Notes

Version 1.7.1

Added Features

  • Added the ability to select sub-meshes based on conical regions
  • Added the ability to perform sculpting by moving points
  • Added the ability to locally smooth point in a mesh

Fixed Bugs

  • Sometimes the contour surface selection will miss some points on the surface and in the result, when moving the selected surface point, those missing points won't get moved. (Mantis: 9612)

Version 1.6.4

Added Features

  • Change default color-by of mesh to use Region
  • After extraction, filters panel is shown and subset is selected
  • Change the subset name to Sel(N) instead of Subset
  • Moved the Visibility column to the right of the tree
  • Added a column and an icon to show Active input
  • Added an Extract_Subset tool button in selection toolbar
  • Added SelectAll and ClearSelection buttons
  • Replaced SetActive and Delete with context menu
  • New Shortcuts: E for Extract, A for Activate as input
  • Added Warning for trying selection while the active input is invisible
  • After selection, the selection panel is shown

Fixed Bugs

  • Wrong cell material IDs were changed when dealing with an inverted selection
  • Fixed that Select All does not allow to change the region Id (Mantis: 9592)
  • Wrong selection being displayed when changing the material ID of inverted selection (Mantis: 9694)

Version 1.6.3

Fixed Bugs

  • Extract selection is enabled now after an inversion of selection is performed (Mantis bug# 0009232)
  • Fixed the bug that extracted selection sometimes not showing correctly after being set as the active input (Mantis bug# 0009231)

Version 1.6.2

Added Features

  • Added options on the UI to turn on/off labels for selected mesh cells/points ID.

Version 1.6.1

Added Features

  • Added new features that all the subsets have their own representations, which are linked with display panel if selected; also visibility of these
    representations can be controlled directly within the list of inputs with the eyeball icons.

Version 1.6.0

Added Features

  • Added support for extracting selection as new input to filters ---

    (Known issue: Invert selection of Paraview's FRUSTUM selection does not work properly, so when selection is inversed, the changeMeshMaterialId and extractSelectionAsInput are disabled)

    (Note: After selection is extracted, a new "subset" will be added to the list in "Filters" tab. In order to set this new subset as filters input, it has to be selected in the list, then click on "Set Selected as Input" button on the Filters tab)

Fixed Bugs

  • Fixed issues of disconnecting with a server in client-server mode.
  • Fixed issues with updating histogram and filters after selection.

Version 1.5.0

Added Features

  • This is the first release of Mesh Viewer application.

    This application can load a 2dm or 3dm mesh file, and automatically connects the reader with ThresholdFilter->MeshQualityFilter->HistogramFilter, so that the mesh can be examined with various statistics. Also the materialIds of the cells in the volume mesh can be modified by selecting the cells then assigining a new Id and the modified mesh can be saved saved out.

  • The cliet-server mode of the application is turned on, but with limited testing, so it may be unstable in Client-Server mode.
  • Since this is the first time this application is shown to the users, feedbacks will be very much appreciated.
