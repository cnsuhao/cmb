CMB Suite Applications Release Notes

Version 1.7.1

Added Features

  • All Applications can now display the current working filename in the title window
  • Added support for writing Gambit files
  • Added read write support for PT123 files
  • Added support for 2D Poly Files
  • Added support for Borehole Files (.bor)
  • Added support for writing out Hot Start Files
  • Enhanced Reading Functionality got CMB Mesh Files
  • Added Context Menu Support in the 3D render window (triggered by right mouse button click
  • Added new stream tracer filter in CMB ParaView Plugin
  • Added new filter for agent-based simulationms that results in the creation of streamlines for the agents

Fixed Bugs

  • Fixed click recording for tree widgets in ParaView

Version 1.6.4

Added Features

  • Changed direction for camera dolly movement to be the same as zoom
  • Added Global Selection Shortcut to Render Window ("S" Key)
  • Added Global Reset Camera to Render Window (SpaceBar)
  • Change Display as outline limit to 2.5 million cells
  • LIDAR Point Reader is more robust. The reader will properly read in files that has the number of points on the first non-empty line and will properly error out when the number of points is missing

Fixed Bugs

  • Fixed bug in reading binary point files w/r to CMB Applications and ParaView Plugin
  • Fixed Open Shortcut to be Ctrl+O/Meta+O
  • Corrected Save Dialogs becoming hidden when typing in file name on OSX

Version 1.6.2

Added Features

  • Added Camera Dialog to all CMB Suite applications
